# Ontology Vision: Acme Supply Chain Control Tower

## Core Entity Set

- Primary operational entity
- Capacity / resource entity
- Constraint or risk entity
- Actionable workflow state entity

## Key Relationships

- Operational entity depends on resource entity
- Risk entity escalates through workflow state
- Action path updates the real-world operating picture

## Rollout View

- Phase 1: core operational picture
- Phase 2: decision support and escalation
- Phase 3: automation and optimization
