# Examples

These examples are synthetic but operator-realistic. They exist to show what a complete artifact chain looks like when `pstack` is used with discipline.

## Engagements

- `engagements/acme-supply-chain/`: supply chain / manufacturing control-tower example
- `engagements/northstar-healthcare/`: healthcare patient-flow example

Each engagement includes a full artifact set matching the filenames defined in `conductor.json`.
