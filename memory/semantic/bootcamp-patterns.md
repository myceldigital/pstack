# Bootcamp Patterns

## Durable Patterns

- The best headliner use cases are anchored to one daily operational decision, one named operator, and one measurable metric.
- The bootcamp collapses when data ownership is vague, even if the systems are technically reachable.

## Common Failure Modes

- “Visibility” is treated as the use case instead of the decision workflow.
- Too many use cases are allowed to survive day one.
