# Evals

`pstack` evals are artifact- and skill-oriented, not prompt-only. Each case points at a realistic example engagement, defines the target skills, and captures what a strong output should contain.

## Layout

- `ARTIFACT-RUBRICS.md`: acceptance rubric for every named artifact
- `SKILL-SCORECARD.md`: benchmark coverage and target scores by skill
- `cases/`: concrete benchmark cases with rubric and scorecard files
- `golden/`: expected-output structure references for key cases
